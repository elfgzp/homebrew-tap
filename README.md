# Homebrew Tap for Anime1 Desktop

This is the official Homebrew tap for [Anime1 Desktop](https://github.com/elfgzp/anime1-desktop), an anime browser application.

## Installation

```bash
# Add the tap
brew tap anime1-desktop/tap

# Install anime1
brew install --cask anime1
```

## Updating

```bash
brew update
brew upgrade anime1
```

## Uninstalling

```bash
brew uninstall anime1
brew untap anime1-desktop/tap
```
